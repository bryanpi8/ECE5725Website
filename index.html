November 29, 2024

Week 3&4 update: GUI and Hardware testing

This week was focused on hardware testing and GUI development. We tested the heating coil for temperature increase and found that it is very significant, it may stay cooler than 100C with indefinite heating but so far in our tests it rose to 75C over a few minutes of off and on use. This shows that we may need to implement a maximum time for holding the sample at the desired temperature, this likely could be implemented as a coil on time that is tracked in the python script. To measure these temperatures we used the infrared thermometer that we plan to use in the project, serving as its own independent test, wired to the pi and sending data via I2C we had accurate data with a decent refresh rate.

We also dove into the GUI development, creating a fully functional touch GUI that will allow the user to control the temperature effectively. One thing that is still missing is a stop button, however it may be an unnecessary addition to the program given that the sample can be removed at any time.

November 12, 2024

Week 2 update: Deep Dive

In this project, we aim to be able to successfully heat a small sample of metal to a chosen temperature. We aim to not overshoot this temperature and also ensure that the sample maintains this temperature for a predetermined amount of time. This will be achieved using the Raspberry Pi as a controller. Timeline: 11/19 Functioning Heater, All necessary parts ordered 11/26 Functioning Laser Thermometer 12/3 Ability to drive heater with Raspberry Pi 12/10 Final Design In addition to the Raspberry Pi, we will be using a purchased induction heater circuit including the heater coil, a laser thermometer, and a breadboard equipped with a 5V relay for controlling the induction heater circuit. For software, it makes the most sense to code this in python so we can have a loop of code continuously running to monitor sample temperature. The code will be much less involved than the hardware and will consist of taking the readings from the laser thermometer and varying the output of a GPIO pin using PWM. This project functions utilizing the Raspberry Pi as an embedded device. Although the induction heater itself can simply be powered with a standard DC power supply, our project differs in that we want to monitor and control the temperature of our sample. In order to accomplish this some form of embedded device is necessary in order to run code to perform this control.
